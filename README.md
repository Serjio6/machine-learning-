# Purple Cap Prediction using Support Vector Machine (SVM)

## Project Overview
This project predicts the potential Purple Cap winner in the IPL using a Support Vector Machine (SVM) machine learning model. The model is trained on IPL player performance data to identify patterns and make predictions.

## Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Support Vector Machine (SVM) model
- Model evaluation
- Saved trained model (`model.pkl`)

## Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Pickle

## Project Files
- `Purple_Cap_Prediction.ipynb` – Google Colab notebook
- `model.pkl` – Trained SVM model
- `dataset.csv` – IPL dataset
- `README.md` – Project documentation

## Model Workflow
1. Load the IPL dataset.
2. Clean and preprocess the data.
3. Split the dataset into training and testing sets.
4. Train the Support Vector Machine (SVM) model.
5. Evaluate the model performance.
6. Save the trained model as `model.pkl`.
